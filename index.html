dnsxd

dnsxd is an authoritative DNS server written in Erlang which supports:

  • Apple Wide-Area Bonjour Extensions:
    • DNS Long Lived Queries
    • Update Leases
  • DNS Update
    • Mandatory TSIGs
    • Optionally restricts clients to manipulating only DNSSD records
  • DNSSEC
    • NSEC3 white lies
    • RSA SHA1 signatures with SHA2 coming shortly
  • Modular Backends:
    • CouchDB via CouchBeam

It has no release package or documentation at present. Source code is hosted on GitHub and is available under the Apache 2 license. Inquiries.
